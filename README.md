## 💼 Sobre o projeto

Este projeto foi desenvolvido para transformar o link da bio em uma **ferramenta de conversão**, não apenas uma lista de links.

A proposta é simples: direcionar o cliente para ações que geram resultado.

## 🛠️ Como rodar localmente

### 1. Clonar repositório

```bash
git clone https://github.com/seu-usuario/link-bio-template.git
```

### 2. Entrar na pasta

```bash
cd link-bio-template
```

### 3. Instalar dependências

```bash
npm install
```

### 4. Rodar o projeto

```bash
npm run dev
```
